## 🖥️ Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/Rajni-yadav146/portfolio_website.git
    cd portfolio_website
    ```

2. **Install dependencies**
    ```bash
    npm install
    ```

3. **Start the development server**
    ```bash
    npm start
    ```

4. **Open in browser**
    - Visit `http://localhost:3000`
